Hot Standby Router Protocol: HSRP

RFC2281 (www.faqs.org/rfcs/rfc2281.html)

Constants

HSRP_INITIAL = 0
HSRP_LEARN = 1
HSRP_LISTEN = 2
HSRP_SPEAK = 4
HSRP_STANDBY = 8
HSPR_ACTIVE = 16

Attributes

group    Standby group
hellotime    Time between the hello messages that this router sends. Obviously only useful in hello messages
holdtime    Length of time that this hello message should be considered valid. Obviously only useful in hello messages.
opcode    Type of the HSRP message contained in this packet.
password    Clear-text, 8-character reused password. Defaults to ‘cisco‘
payload    Payload. Generally unused.
priority    Priority used to determine active and standby routers. Higher priorities win, but a higher IP address wins in the event of a tie.
reserved   
state    Current state of the router sending the message
version    Version of the HSRP message contained in this packet. Defaults to 0
vip    Virtual IP address used by this group
